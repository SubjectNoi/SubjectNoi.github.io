SM Programming Guide

Posted on 2019-05-18

这一篇是Volta SM Programming Guide的自翻，渣水平见谅，由于Ampere还没正式发布，就不放最新的了。

概览

术语

CTA

CTA (Cooperative Thread Array)是一组具备对称的执行、访存性能的线程。CTA必须支持1-1024之间的任意数目的线程。

Quad

Quad是一组具有特殊局部性的4个线程：这四个线程在屏幕空间的2个维度(X与Y)相关联，以允许通过有限元素差分的方式计算该点导数。Quad是片段处理(Fragment Processing)的基本单元。

Primitive

Primitive可被解释为实体，一个实体是由一系列顶点组织而成的图形。一个实体是顶点处理和几何处理的基本单元。

Patch

Patch是一组能够参数化描述几何图形的控制点。Patch本身被分为线、三角形、多边形，Patch是光栅操作(Tessellation Task)的基本单元。

皇牌空战7：未知天空(ACE7: Skies Unknown) 武器Tips

Posted on 2019-02-26

首先评论一下这个游戏，我不知道为什么IGN只给打7分，给我打我真的打8.5-9。这个游戏属实太戳我了，不管是剧情、关卡设计还是玩法。
辣么不说废话了，武器Tips目的主要在于：

  • 推荐一下我觉得好用的武器
  • 太久没更新博客
    辣莫武器Tips还是从单人和多人入手好了。在此之前，稍稍介绍一下本作我已经解锁了的武器。此处以对空/对面/皆可做区分。

    General

    MSL

    那么这个就是鹰击长空系列中所谓的JSM(Joint Strike Missile)，所谓联合攻击导弹既可以攻击空中单位也可以攻击面上单位，然而现实很残酷，不存在这样的导弹。这种导弹是玩家在游戏中发射机会最多的导弹，标准锁定距离2000米，因制导性能不佳，发射时建议咬尾时发射，务必不要再敌机9点钟和3点钟方向发射，TU-160都能躲掉。对于正面遭遇，建议避免大攻角发射。导弹威力较小，基本只有轻型战斗机比如米格，F16这种标准的高低搭配轻型战斗机可以一发击落，重型战斗机比如SU系，攻击机等需要2-4发。配件建议携带加快导弹飞行速度，那个加制导能力的聊胜于无。

    PLSL

    激光机炮，高难度打无人机（真他妈不是人打的）狗斗时候有奇效。19关武库鸟SU-57+PLSL苏联歼星舰了解一下。

    EML

    电磁炮，摸奖武器，打船一流，有效距离10000m，装填较慢，打的时候建议减速，有几关要刷分的一定要带，可以在舰载机还未起飞的情况下直接击沉航母。

    TLS

    看上去很牛逼的武器，但是用起来真的不行，有效距离6000m，最要命的是电磁炮3发解决一艘航母这个扫2发还扫不爆一艘航母，燃用想用这个打无人机的也不用想了，真的不好用。

    MGP

    机炮吊舱，这个武器也算机炮，所以可以用来完成机炮打飞机的任务。

    Anti-Air:

    SAAM

    半主动雷达制导空空导弹，这个导弹在鹰击长空2里面锁定距离是10000m，ACE7锁定距离6000m，制导能力很不错，只要目标保持在雷达照射范围内。

    SASM

    接近目标后如果无法直接命中将会空爆的导弹，本身制导能力一般，但是可以磨削，锁定距离4000m。

    HPAA

    高威力空空导弹，专门用来打TU-160/TU-95/A-10/SU-34这些的，不多说了。

    LAAM

    超远距离空空导弹，说是超远，但是还是没有电磁炮远，目前体感无人机在距离远的时候动作幅度会比较小，这时候用LAAM打命中率会比较高。

    HVAA

    高速空空导弹，目前个人没发现什么很好用的用法。

    HCAA

    QAAM

    制导能力较强，威力也大，加上特殊导弹制导插件后制导能力极强，多人中会打一点的很多都会带这个导弹。

    4/6/8 AAM

    ACE7 中多联装空空/空面导弹的锁定方式（火控雷达算法）与鹰击长空中不太一样。鹰击长空中多联装AAM每个目标锁定一发，故可以等目标机全部进入火控雷达照射范围后发射。而ACE7则为锁定已经在照射范围中的敌机，并短时间内将这些目标分配给4/6/8枚导弹，单个目标可被锁定4（不知道是不是至多4）枚导弹。然而这个多联装空空导弹的制导能力真的很差，15关开头就算是占据高海平面从后上方往下锁定敌机，被3枚多联装空空导弹锁定的幻影2000-5都能随便逃走，而且发射一次8发就没了，真的很浪费，要避免浪费只能刚锁定1-2个目标的时候就击发，那我还用你这个导弹干啥呢？？

    Anti-Surface:

    本节中所有的空面导弹的末端制导方式与一般导弹是不一样的，其机动方式平飞至目标上空立即调整姿态至高角度俯冲。

    LASM

    LACM

    LAGM

    UGB

    无引导炸弹，载弹量多，范围大，可以第六关用用。

    RKT

    RKT写的40！不是说只有40发！意思是能发射40*8发，射完8发算一次，总共能射40次。

    GRKT

    GPB

    SFFS

    SOD

    F-35配置的航弹，对投射方向较为严苛，因杀伤范围是投射方向的延伸，故需要机鼻正对目标簇方向时投射较好。

    XSBD

    可以说是目前我感觉最好用的空面武器了，只有F-22有，锁定发射，炸弹带引导，威力大。极其顺手。

    4/6/8 AGM

    多联装空面导弹，锁定方式和多联装空空导弹类似。不过因为用到这种导弹的时候都是地面目标很多很密集的时候，所以不会出现一个地面雷达站锁3-4枚导弹的情况，用起来比多联装空空导弹好用。

Constructing CUTLASS with Windows 10 and CUDA 10.0

Posted on 2019-01-28 | In Constructing environment

About CUTLASS

CUTLASS is a template library developed by Nvidia with CUDA 10.0 aiming to deal with Linear Algebra Computation. You can find CUTLASS here. The readme.md of the original project is based on unix family OS, This article will introduce how to construct CUTLASS on windows 10.

Parperation

  • Windows 10 64-bit
  • CUDA 10.0
  • Visual Studio 2017
  • CMAKE (Version > 3.10)
  • git

Please make sure all the CUDA component are in the system path, and test with related command: nvcc, cl, etc.

Constructing CUTLASS
